Это демонстрационный модуль для портфолио. Он показывает подходы к разработке на Bitrix Framework: D7 ORM, агенты, компоненты, сервис-локатор, работа с внешним API.

Модуль для **1С-Битрикс / Битрикс24**: автоматическое получение, хранение и отображение курсов валют с сайта Центрального Банка РФ.

---

## Возможности

- ✅ Получение курсов валют через официальное API ЦБ РФ (XML)
- ✅ Хранение истории курсов в собственной таблице (D7 ORM)
- ✅ Автоматическое обновление через агент (раз в сутки)
- ✅ Автоматическая очистка старых данных (срок хранения настраивается)
- ✅ График динамики курса за выбранный период
- ✅ Административный интерфейс для настройки
- ✅ Компонент для вывода списка курсов на сайте

---

## Установка

### 1. Скопировать модуль

Скопируйте папку `local.currencies` в `/local/modules/` вашего сайта.

### 2. Установить через админку

Перейдите в **Настройки → Marketplace → Установленные решения**. Выберите модуль `local.currencies` и установите его.

### 3. Проверить установку

После установки:
- В таблицах БД появится `loc_curr_currency_rate`
- В административном меню появится пункт **«Курсы валют»**
- Будет добавлен агент `Local\Currencies\Agent\UpdateRatesAgent::run();`

---

## Настройка

Перейдите в **Настройки → Настройки продукта → Настройки модулей → Курсы валют**.

| Параметр | Описание | Значение по умолчанию |
|----------|----------|----------------------|
| **Срок хранения курсов (дней)** | Курсы старше этого количества дней будут автоматически удаляться | `30` |

## Использование

### Компонент для вывода списка курсов

```php
<?$APPLICATION->IncludeComponent(
    "local.currencies:rates.list",
    ".default",
    [
        "CACHE_TIME" => 3600,
        "LIMIT" => 10,
    ]
);?>
```
### Компонент для отображения графика

```php
<?$APPLICATION->IncludeComponent(
    "local.currencies:rates.chart",
    ".default",
    [
        "CURRENCY" => "USD",
        "DAYS" => 30,
    ]
);?>
```
### Административная страница с графиком

После установки модуля перейдите в административный раздел: Настройки → Курсы валют → График курсов.
